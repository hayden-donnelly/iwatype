## About

Iwatype is an *aesthetic* typing app.
The *aesthetic* is a work in progress.

## Dependencies
* Node
* Express
* SQLite3
* Git LFS

## Git LFS
This project uses Git LFS to store its databse.
Make sure you have it setup for your account and machine with the following command:
```
git lfs install
```
